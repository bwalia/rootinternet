How It Works

Getting Started & Configuration

Getting started with Root Internet Pvt Ltd is easy. Using our intuitive Web-based control panel, enter the URL you wish to monitor (e.g. http://www.yourdomain.com), and select the monitoring interval (e.g. 1 minute). The monitoring interval represents how often we’ll check to see if the website is online and responding properly. The smaller the interval, the more frequently we’ll check your website.
